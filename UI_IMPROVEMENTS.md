## 🎨 ЧТО ИЗМЕНЕНО

### 1. ✅ Отображение статусов в недоступных ячейках

**Было:** Просто серые закрашенные ячейки

**Стало:** Текст с указанием причины недоступности:

- `ВІДПУСТКА` - если статус VACATION
- `ВІДРЯДЖЕННЯ` - если статус TRIP
- `ЛІКУВАННЯ` - если статус SICK
- `ІНШЕ` - если статус OTHER
- `ВІДПОЧИНОК` - день отдыха до/после статуса (restBeforeStatus/restAfterStatus)
- `ЗАБЛОКОВАНО` - если день недели заблокирован (blockedDays)
- `ПІДГОТОВКА` - день подготовки перед статусом (PRE_STATUS_DAY)

---

### 2. ✅ Иконка руки для ручных назначений

**Было:** `НАРЯД 🔒` для всех назначений с замком

**Стало:**

- `НАРЯД ✋` - если назначение ручное (type='manual') и не заблокировано
- `НАРЯД 🔒` - если назначение заблокировано (isLocked=true)
- `НАРЯД` - если автоматическое (type='auto') и не заблокировано

## 🔍 КАК ПРОТЕСТИРОВАТЬ

### 1. Запустить приложение:

```bash
cd /Users/vladyslav/Projects/varta-2026
npm run dev
```

### 2. Проверить отображение статусов:

**Тест 1: Отпуск**

1. Перейти в "Бойці"
2. Выбрать пользователя → Редагувати
3. Установить статус "Відпустка"
4. Указать даты (statusFrom, statusTo)
5. Зберегти
6. Перейти в график → должен показывать "ВІДПУСТКА" в ячейках

**Тест 2: Отдых до/после**

1. В модалке редактирования пользователя
2. Включить "День відпочинку до" или "День відпочинку після"
3. Зберегти
4. В графике должны появиться "ВІДПОЧИНОК" в соответствующих днях

**Тест 3: Заблокированные дни**

1. Редагувати пользователя
2. Включить чекбокс блокировки для конкретного дня (например, суббота)
3. Зберегти
4. В графике должен показывать "ЗАБЛОКОВАНО" во всех субботах

### 3. Проверить иконки:

**Тест 4: Ручное назначение**

1. В графике кликнуть на пустую ячейку
2. Выбрать пользователя (ручное назначение)
3. Должно показать "НАРЯД ✋"

**Тест 5: Блокировка назначения**

1. Кликнуть на существующее назначение
2. Выбрать "Заблокувати" (замок)
3. Должно показать "НАРЯД 🔒"

**Тест 6: Автоматическое назначение**

1. Использовать автогенерацию графика
2. Автоназначения должны показать просто "НАРЯД" (без иконок)
