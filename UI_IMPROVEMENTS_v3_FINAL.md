## 🐛 ИСПРАВЛЕННЫЕ ОШИБКИ

### Ошибка 1: День ДО командировки не помечался ❌

**Проблема:**  
День перед командировкой (restBeforeStatus) не показывал "ЗВІЛЬН. ВІД ЧЕРГ."

**Причина:**  
Неправильный порядок проверок в `availability.utils.ts`:

```typescript
// БЫЛО (неправильно):
if (dateStr >= from && dateStr <= to) return 'STATUS_BUSY'; // сначала
if (u.restBeforeStatus && u.statusFrom) return 'REST_DAY'; // потом (не доходило)
```

**Решение:**

```typescript
// СТАЛО (правильно):
// Сначала проверяем дни до/после
if (u.restBeforeStatus && u.statusFrom) return 'REST_DAY';
if (u.restAfterStatus && u.statusTo) return 'REST_DAY';
// Только потом проверяем основной период
if (dateStr >= from && dateStr <= to) return 'STATUS_BUSY';
```

✅ **Теперь работает!**
