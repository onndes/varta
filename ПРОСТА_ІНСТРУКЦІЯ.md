```bash
cd /Users/vladyslav/Projects/varta-2026
npm run build
```

✅ **Результат:** в папці `dist/` з'явиться файл `index.html`

---

## 🚀 Використання (ДУЖЕ ПРОСТО!)

### Для користувача (двічі клікнути):

1. Скопіюйте файл `dist/index.html` на флешку або відправте по email
2. **Просто двічі клацніть** по файлу `index.html`
3. Він відкриється в браузері — **ВСЕ!** ✅

**Ніяких серверів, ніяких налаштувань — просто відкрити файл!**

---

## 💾 Збереження даних

Всі дані зберігаються **в браузері автоматично**.

### Переносення даних:

1. **Експорт:** У застосунку натисніть кнопку "Експорт даних"
   - Збережеться файл `VARTA_BACKUP_[дата].json`

2. **Імпорт:** На іншому ПК натисніть "Імпорт даних"
   - Виберіть файл JSON

**⚠️ ВАЖЛИВО:** Робіть бекап мінімум раз на тиждень!
