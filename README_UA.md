# 🛡️ VARTA-2026 — Система автоматичного розподілу нарядів

PWA застосунок для управління добовими чергуваннями в військових підрозділах.

## 🚀 Швидкий старт

### Розробка (Development)

```bash
npm install
npm run dev
```

### Оффлайн розгортання (Production)

#### Крок 1: Збірка проекту

```bash
npm run build
```

#### Крок 2: Запуск локального сервера

**macOS / Linux:**

```bash
./start-offline.sh
```

**Windows:**

```bash
start-offline.bat
```

**Або вручну (Python):**

```bash
cd dist
python3 -m http.server 8080
```

Відкрийте: http://localhost:8080

## 📦 Структура проекту

```
varta-2026/
├── src/                    # Вихідний код
│   ├── components/         # React компоненти
│   ├── services/           # Бізнес-логіка
│   ├── hooks/              # Custom hooks
│   ├── types/              # TypeScript типи
│   ├── utils/              # Допоміжні функції
│   └── styles/             # SCSS стилі
├── public/                 # Статичні файли
├── dist/                   # Зібрана версія (після npm run build)

```

## 🔧 Технології

- **React 19** + TypeScript
- **Dexie.js** (IndexedDB wrapper)
- **Bootstrap 5.3**
- **Vite** (build tool)
- **PWA** готовність

## 🗄️ База даних

**Важливо:** Дані зберігаються локально в браузері (IndexedDB).

### Резервне копіювання

1. Експорт: кнопка "Експорт даних" → збережеться JSON файл
2. Імпорт: кнопка "Імпорт даних" → виберіть JSON файл

**Рекомендація:** Робіть бекап мінімум раз на тиждень.

## 🎯 Призначення

Система для малих підрозділів (2-100 осіб):

- Автоматичне планування добових чергувань
- Справедливий розподіл навантаження
- Врахування відпусток, відряджень, хвороб
- Блокування днів тижня
- Система боргів і компенсацій
