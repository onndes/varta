## 🎨 ЧТО ИЗМЕНЕНО

### 1. ✅ Отображение статусов в недоступных ячейках

**Было:** Просто серые закрашенные ячейки

**Стало:** Текст с указанием причины недоступности:

- `Відрядження` - командировка (TRIP)
- `Відпустка` - отпуск (VACATION)
- `Лікування` - лечение (SICK)
- `Інше` - другое (OTHER)
- **`ЗВІЛЬН. ВІД ЧЕРГ.`** - **освобождён от дежурства** (день отдыха)
- `ЗАБЛОКОВАНО` - день недели заблокирован

**"ЗВІЛЬН. ВІД ЧЕРГ." означает:**

- День перед командировкой/отпуском (если restBeforeStatus = true)
- День после командировки/отпуска (если restAfterStatus = true)
- Сотрудник освобождён от чергування в эти дни

---

### 2. ✅ Замок только для заблокированных

**Было:** Рука ✋ для ручных, замок 🔒 для заблокированных

**Стало:**

- `НАРЯД 🔒` - заблокированное назначение (isLocked=true)
- `НАРЯД` - обычное назначение (ручное или автоматическое)

## 📊 ТАБЛИЦА СТАТУСОВ

| Код              | Текст                               | Значение                  |
| ---------------- | ----------------------------------- | ------------------------- |
| `STATUS_BUSY`    | Відрядження / Відпустка / Лікування | В период статуса          |
| `REST_DAY`       | **ЗВІЛЬН. ВІД ЧЕРГ.**               | Освобождён от дежурства   |
| `PRE_STATUS_DAY` | **ЗВІЛЬН. ВІД ЧЕРГ.**               | Подготовка перед статусом |
| `DAY_BLOCKED`    | ЗАБЛОКОВАНО                         | День недели заблокирован  |
