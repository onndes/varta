# 🛡️ VARTA-2026 — Швидкий старт

## 📦 Збірка для оффлайн використання

### 1️⃣ Збудуйте проєкт

```bash
cd /Users/vladyslav/Projects/varta-2026
npm run build
```

✅ Результат: папка `dist/` з усіма файлами застосунку

### 2️⃣ Запустіть локально

**macOS / Linux:**

```bash
./start-offline.sh
```

**Windows:**

```bash
start-offline.bat
```

**Або вручну (Python):**

```bash
cd dist
python3 -m http.server 8080
```

Відкрийте: **http://localhost:8080**

---

## 📁 Перенесення на інші комп'ютери

### Варіант А: Копіювання папки

1. Скопіюйте папку `dist/` на флешку
2. На іншому ПК: відкрийте `dist/index.html` у браузері

### Варіант Б: ZIP архів

```bash
zip -r varta-2026-offline.zip dist/
```

Відправте ZIP → розпакуйте → відкрийте

## 💾 Синхронізація даних

**Експорт:** кнопка "Експорт даних" → збережеться JSON  
**Імпорт:** кнопка "Імпорт даних" → виберіть JSON

⚠️ **Важливо:** Робіть бекап мінімум раз на тиждень!

## 🌐 Робота в локальній мережі

```bash
# На головному ПК
python3 -m http.server 8080

# Дізнайтесь IP адресу
ifconfig  # macOS/Linux
ipconfig  # Windows

# Інші ПК підключаються:
# http://192.168.x.x:8080
```
