## 📁 Крок 2: Що передати користувачу

### Варіант А: Тільки файл (найпростіше)

Передайте файл:

```
dist/index.html
```

### Варіант Б: З інструкцією (рекомендується)

Передайте:

```

dist/ІНСТРУКЦІЯ.txt
```

### Варіант В: ZIP архів

```bash
cd /Users/vladyslav/Projects/varta-2026/dist

```

Передайте файл `varta-2026.zip`

## ⚠️ ВАЖЛИВО: Попередьте користувача

✅ **Робити бекап мінімум раз на тиждень** (кнопка "Експорт даних")  
✅ **Не видаляти cookies** в браузері  
✅ **Не використовувати режим "Інкогніто"**  
✅ **Зберігати файли JSON** з експорту

❌ **НЕ працює:** якщо очистити дані браузера  
❌ **НЕ працює:** в приватному режимі

---

## 🔧 Якщо щось не працює

### Проблема: "Не відкривається"

→ Відкрити в іншому браузері (Chrome, Firefox)

### Проблема: "Дані зникли"

→ Імпортувати з останнього бекапу JSON

### Проблема: "Не зберігає"

→ Перевірити що не використовується режим "Інкогніто"
